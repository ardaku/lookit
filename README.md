Rust library for asynchronously connecting to devices

## Currently Supports
 - Linux (via inotify)
 
## Planned Support
 - Web / Wasm (via JS promises)
 - MacOS / iOS
 - Windows
 - Redox
 - Quantii / Daku
 - Android
 - ...and more!
