# Lookit!
Rust crate to asynchronously get device connection events.

## Currently Supports
 - Linux (via inotify)
 
## Planned Support
 - Web / Wasm (via JS promises)
 - MacOS / iOS
 - Windows
 - Redox
 - Quantii / Daku
 - Android
 - ...and more!
